Trips — Data Deletion Instructions

This page explains how users can request deletion of personal data associated with the Trips mobile app. Add this URL to your Facebook App's "User data deletion" field so reviewers and users can see how to delete data.

How to request deletion

To request deletion of your personal data, please contact our support team using the details below. Include the information listed in the sample email so we can find and remove your account quickly.

What we'll remove

  1. User profile information (name, email, profile picture) stored by Trips.
  2. Bookings, reservations, or other records tied to your account (unless legally required to retain some records).
  3. Personal preferences and app-specific metadata.
Important: We will remove personal data from our active systems, but some data (for example, anonymized analytics, cached backups, or records we are legally required to keep) may remain for a short time. We will delete data from backups according to our retention policy.

Third-party data & revoking access

If you signed in with a third-party account (for example, Facebook), you should also revoke app access from your third-party account settings. For Facebook, go to Facebook Settings → Apps and Websites and remove Trips from the list.

Timeframe & confirmation

We will acknowledge your request within 5 business days and aim to complete deletion within 30 days. After deletion is finished we will send a confirmation email to the address you provided in the request.

Hosting this file on GitHub Pages (step-by-step)

  1. Create a new GitHub repository (public is fine) — e.g. trips-privacy.
  2. Create a new file named deletion.html in the repo root and paste the contents of this file into it. Commit the change.
  3. Go to the repository Settings → Pages. Under Source choose the branch (usually main) and folder / (root), then click Save.
  4. Wait a minute. Your page will be available at:
    https://<your-github-username>.github.io/trips-privacy/deletion.html
  5. Copy that URL and paste it into Facebook Developers → your app → Settings → Basic → User data deletion (Data deletion instructions URL field).
